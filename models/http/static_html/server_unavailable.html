Server Unavailable

The TrouxSource server cannot be accessed for one of the following reasons:

  • The server is down. Contact your system administrator.

  • The Marshalling object is not configured correctly. Right-click the Marshalling object and specify the server name.

  • The login was cancelled. To display the Login dialog box again, go to any other tab in the left pane and then return to TrouxSource tab.

  • Your login ID does not have access to the TrouxSource server. Contact your system administrator.

  • If forms-based authentication is being used, the required fields on the Network tab of the Preferences dialog box are incorrect. See the online help for details.
